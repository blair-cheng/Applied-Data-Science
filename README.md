**Highlights:**
- RAG pipeline final project: PDF ingestion → recursive text splitting → OpenAI embeddings → Pinecone vector store → MMR retrieval → GPT-3.5 with anti-hallucination prompt engineering — the exact architecture pattern behind enterprise document Q&A systems
- PCA relative importance analysis with four metrics (last, first, betasq, pratt) for principled dimensionality reduction
